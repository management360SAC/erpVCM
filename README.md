# VCM ERP (MySQL) 🚀

ERP web para empresas orientado a **gestión comercial y operativa**, con módulos como:
- **Pipeline / Embudo de ventas**
- **Leads**
- **Alertas y Recordatorios**
- **Cotizaciones** (Nueva cotización + Historial)
- **Servicios / Pagos**

> Proyecto fullstack con **MySQL** como base de datos y un **frontend React + Vite + TypeScript**.

---

## 🧱 Tecnologías

### Frontend
- React + Vite
- TypeScript
- Router (rutas tipo `/pipeline/leads`, `/cotizaciones/nueva`, etc.)
- Layout y navegación centralizada en `src/layout/AppLayout.tsx`

### Backend
- API REST (estructura en carpeta `/backend`)
- Conexión a **MySQL**
- Arquitectura modular (recomendado: controllers / services / repositories)

### Base de datos
- MySQL (scripts y/o definición en `/db`)
